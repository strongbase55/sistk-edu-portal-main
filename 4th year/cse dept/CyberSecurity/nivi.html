Future Generations Of Malware detection
Using genetic Algorithm

This project used for detecting rare classes of malware as well as possible and potential future mutations caused by metamorphic engines. Many of the malware classes we focused on are extremely rare and obfuscated. We were able to collect and acquire some relevant data for these classes from real time datasets. The method utilizes genetic algorithms (GA),aiming to improve detection accuracy.
